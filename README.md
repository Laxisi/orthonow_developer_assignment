# OrthoNow Developer Assignment

This is my submission for the Developer Assignment.

- **task1/README.md** - my plan for tracking events on OrthoNow's website using GTM
- **task2/index.html** - the landing page I built for the "Book a Consultation" campaign (just open the file in a browser, no setup needed)
- **task3/README.md** - my answer on how I would connect the landing page to HubSpot and WhatsApp

## How to try out the landing page

1. Open `task2/index.html` in any browser.
2. Type your name and a 10-digit phone number in the form.
3. Click "Book Consultation". The page will show a thank-you message, without reloading.
4. Right click on the page and choose "Inspect", then click the "Console" tab.
5. In the console, type `dataLayer` and press Enter - you should see the event that got sent when you submitted the form.
