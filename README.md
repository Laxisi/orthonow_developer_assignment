# OrthoNow Developer Assignment

This repo contains my submission for the Developer Assignment.

- **task1/README.md** - GTM event tracking plan for OrthoNow's website
- **task2/index.html** - Landing page for the "Book a Consultation" campaign (open the file directly in a browser, no setup needed)
- **task3/README.md** - My answer on how I'd connect the landing page to HubSpot and WhatsApp

## How to test the landing page

1. Open `task2/index.html` in any browser.
2. Open the browser console (right click > Inspect > Console tab).
3. Fill in the name and a 10-digit phone number, then click "Book Consultation".
4. Type `dataLayer` in the console and press Enter, you should see the event that was pushed.
5. The page will show a thank-you message without reloading.
